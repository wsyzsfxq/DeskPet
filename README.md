# DeskPet

Desktop pet and lightweight AI companion project.

## Concept

DeskPet is a desktop-native pet and companion experience. It is not meant to feel like a traditional game window that the user opens and closes. Most of the time, only the pet stays visible on the desktop, quietly living around the user's actual computer work.

The nest, furniture, care panels, settings, inventory, and other UI can be hidden. With one action, the user can bring that hidden UI back when they want to feed, decorate, configure, or play more actively.

The pet can move around the desktop, react to mouse interaction, interact with its nest and furniture, and later, with explicit user permission, understand what the user is doing on the desktop to provide quiet companionship or assistant-like responses.

## Core Idea

The pet does not live inside a game window. It lives on the user's desktop.

DeskPet should feel like a small digital creature sharing the computer with the user: sometimes playful, sometimes quiet, sometimes curious about what the user is doing. It can be interacted with directly by clicking or gesturing with the mouse, but it should not constantly interrupt the user's work.

## Key Features

- Desktop-resident pet: the pet can stay visible while the rest of the UI is hidden.
- One-action UI reveal: the nest, furniture, care UI, and settings can be summoned back quickly.
- Nest and furniture interaction: the pet can use, inspect, sleep in, or play with items in its living space.
- Direct pet interaction: clicking or using the mouse on the pet can trigger reactions, play, affection, or care behaviors.
- Optional desktop awareness: later versions may let the pet understand what is on screen or what the user is doing, controlled by an explicit on/off switch.
- AI companion layer: the pet may gradually become a lightweight assistant or companion, but the primary interface should remain the pet's behavior, not a plain chatbot.

## Design Tone

Warm, quiet, a little strange, and low-interruption.

The experience should emphasize companionship over productivity pressure. AI features should make the pet feel more aware and alive, not turn it into an intrusive work assistant.
